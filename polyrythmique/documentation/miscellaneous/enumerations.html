BinaryNote

This enum list every note type that can be visually created.

VALUES NOTATION :
All values start by _ due to the fact that a value cannot start by a number.

R -> rest D -> dotted

TR -> triplet

1N -> whole note 2N -> half note 4N -> quarter note 8N -> eight note 16N -> sixteenth note

A number not followed by a N but by an _ specify the number of linked notes after it following themselves. ex : 28N = two eight note or eight note eight note.
